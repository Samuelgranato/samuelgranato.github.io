Como Usar¶

  1. Instalar os pacotes necessários
  2. Criar usuário
  3. Executar softdes.py

Procedimentos

  1. Executar o comando:

    pip3 install flask
pip3 install flask_httpauth
  2. Criar o usuário
    1. Criar um arquivo .csv com o seguinte conteúdo::

      _usuário_,1

      (subtituir _usuário_ pelo nome de usuário desejado)

    2. Executar adduser.py

      Executar o comando:

      python3 adduser.py
  3. Inicializar o servidor

    1. Executar softdes.py:

      python3 softdes.py

    2. Abrir o endereço no navegador. Obs: a senha do usuário será o próprio usuário
